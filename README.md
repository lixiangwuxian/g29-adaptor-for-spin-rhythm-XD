# g29-adaptor-for-spin-rhythm-XD

Write with gpt-4o.

## How to use

```shell
pip install PySDL2
pip install pysdl2-dll
python3 g29_to_xbox_controller.py
```

### In Game

Setting -> Game Setting -> Input -> Control Style -> Select `Rotate Joy Stick`

## Key mapping

|G29|controller|
|---|-----------|
|△|Y|
|□|X|
|○|B|
|×|A|
|dpad|dpad|
|gear down|left bumper|
|gear up|right bumper|
